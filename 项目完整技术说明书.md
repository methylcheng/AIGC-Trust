## 项目概述

**AIGC-Trust** 是一个 AIGC（人工智能生成内容）真实性检测与可信溯源系统，用于识别和验证 AI 生成的图片、视频和文本内容。

### 核心功能

- **多模态检测**: 支持图片、视频、文本的 AI 生成内容检测
- **边缘-中心协同**: 边缘端快速初筛 + 中心端精细复检的两级检测架构
- **国密证书**: 使用 SM2/SM3 算法生成数字证书，确保检测结果不可篡改
- **区块链溯源**: 基于 Merkle Tree 构建内容指纹，实现可追溯性
- **实时任务管理**: 基于 RabbitMQ 的异步任务处理
- **可视化界面**: Vue 3 + Element Plus 构建的现代化前端

### 技术栈

| 类别 | 技术选型 |
|------|---------|
| **后端框架** | FastAPI (Python 3.13) |
| **深度学习** | PyTorch 2.7.1 + CUDA 11.8 |
| **数据库** | SQLite / MySQL |
| **消息队列** | RabbitMQ + Pika |
| **前端框架** | Vue 3 + Vite + Element Plus |
| **加密算法** | 国密 SM2/SM3 (gmssl) |
| **图像处理** | OpenCV, Pillow |
| **视频处理** | FFmpeg (GPU 加速) |

## 代码文件功能说明

### 📁 根目录

#### `main.py` - 后端主入口
- **功能**: FastAPI 应用主程序，注册所有 API 路由
- **启动命令**: `python main.py`
- **端口**: 8000
- **包含路由**:
  - `/api/auth` - 用户认证
  - `/api/contents` - 内容上传与管理
  - `/api/tasks` - 检测任务管理
  - `/api/fingerprints` - 内容指纹
  - `/api/certificates` - 数字证书
  - `/api/audit` - 审计日志
  - `/api/edge` - 边缘节点接口
  - `/api/stats` - 统计数据

#### `start_edge.py` - 边缘检测节点启动脚本
- **功能**: 启动边缘轻量化检测服务
- **特点**: 每 15 帧抽 1 帧，最多 40 帧，不使用深度学习模型

#### `requirements.txt` - Python 依赖清单
- 包含所有后端依赖包及版本要求

#### `requirements-gpu.txt` - GPU 环境依赖
- PyTorch GPU 版本的专用配置

### 📁 core/ - 核心检测引擎

| 文件 | 功能 |
|------|------|
| `image_detector.py` | 图片 AI 生成检测器，加载 ViT 模型进行推理 |
| `video_detector.py` | 视频检测器，抽帧 + GPU 加速预处理 + 融合决策 |
| `video_detector_edge.py` | 边缘轻量化视频检测，启发式快速筛查 |
| `text_detector.py` | 文本 AI 生成检测器 |
| `multimodal_analyzer.py` | 多模态融合分析器 |
| `fusion_decide.py` | 融合决策引擎，综合模型分数、噪声异常、频域特征 |
| `frequency_analysis.py` | 频域分析（DCT、FFT） |
| `video_preprocess_gpu.py` | GPU 加速视频预处理 |
| `watermark_check.py` | 数字水印检测 |

**融合决策权重**:
- 深度学习模型: 80%
- 噪声残差异常: 10%
- 频域特征异常: 10%

## 从零开始启动指南

### 🎯 前置要求

接收项目压缩包后，需要安装以下软件：

1. **Python 3.13** (必须 3.13，不支持 MindSpore)
   - 下载地址: https://www.python.org/downloads/
   - 安装时勾选 "Add Python to PATH"

2. **Node.js 18+** (前端开发)
   - 下载地址: https://nodejs.org/

3. **Git** (可选，用于版本管理)
   - 下载地址: https://git-scm.com/

4. **RabbitMQ** (消息队列，可选但推荐)
   - Windows 安装教程: https://www.rabbitmq.com/install-windows.html
   - 需要先安装 Erlang

5. **FFmpeg** (视频处理)
   - 下载地址: https://ffmpeg.org/download.html
   - 添加到系统 PATH

6. **NVIDIA GPU + CUDA 11.8** (GPU 加速，可选)
   - 如果没有 GPU，系统会自动降级到 CPU 模式

---

### 📦 第一步：解压项目

```bash

cd D:\Projects\AIGC-Trust
```

## 常见问题排查

### ❌ 问题 1: Python 版本不兼容

**错误信息**: `ERROR: No matching distribution found for mindspore`

**解决方案**: 
- 项目已移除 MindSpore 依赖，使用 PyTorch
- 确保 Python 版本为 3.13
- 检查: `python --version`

---

### ❌ 问题 2: 虚拟环境激活失败

**错误信息**: `无法加载文件 .venv\Scripts\Activate.ps1，因为在此系统上禁止运行脚本`

**解决方案**:
```powershell
# 以管理员身份运行 PowerShell
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
```

---

### ❌ 问题 3: RabbitMQ 连接失败

**错误信息**: `ConnectionRefusedError: [WinError 10061]`

**解决方案**:
```powershell
# 检查 RabbitMQ 服务是否启动
Get-Service RabbitMQ

# 启动服务
net start RabbitMQ

# 如果未安装，下载并安装
# https://www.rabbitmq.com/install-windows.html
```

---

### ❌ 问题 4: CUDA 不可用

**检查方法**:
```python
import torch
print(torch.cuda.is_available())  # 应输出 True
print(torch.version.cuda)         # 应输出 11.8
```

**解决方案**:
- 如果没有 NVIDIA GPU，系统会自动使用 CPU 模式
- 如果有 GPU 但 CUDA 不可用，重新安装 PyTorch GPU 版本

---

### ❌ 问题 5: 前端 API 请求失败

**错误信息**: `Network Error` 或 `CORS error`

**解决方案**:
1. 确认后端服务正在运行（http://localhost:8000）
2. 检查 `frontend/vite.config.js` 中的代理配置
3. 清除浏览器缓存并刷新

---

### ❌ 问题 6: 数据库表不存在

**错误信息**: `sqlite3.OperationalError: no such table`

**解决方案**:
```python
# 运行以下脚本初始化数据库
from db.db_conn import engine
from db.models import Base
Base.metadata.create_all(bind=engine)
```

---

### ❌ 问题 7: FFmpeg 未找到

**错误信息**: `FileNotFoundError: ffmpeg`

**解决方案**:
1. 下载 FFmpeg: https://ffmpeg.org/download.html
2. 解压到 `C:\ffmpeg`
3. 添加 `C:\ffmpeg\bin` 到系统 PATH
4. 重启终端验证: `ffmpeg -version`
